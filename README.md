# AvePhysica Notes

This repository hosts the source files for a personal academic notes website built with **MkDocs (Material theme)** and deployed automatically via **GitHub Pages**.

Website:
[https://AvePhysica.github.io/notes/](https://AvePhysica.github.io/notes/)

---

## Overview

* Markdown-based lecture and study notes
* LaTeX math rendering via MathJax
* Structured navigation defined in `mkdocs.yml`
* Automatic deployment through GitHub Actions

All website content is located inside the `docs/` directory.

---

## Workflow

1. Write notes in Markdown under `docs/`.
2. Update navigation in `mkdocs.yml` if needed.
3. Commit to `main`.
4. The site is rebuilt and deployed automatically.

---

This repository serves as a structured archive of academic materials and ongoing study notes.
